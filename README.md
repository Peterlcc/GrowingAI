1. 目前完成项目结构初始化

## 基本功能

- 主页
- 用户注册
- 用户登录
- 用户个人信息修改（仅限昵称、邮箱、密码）
- 添加项目（文件夹上传）
- 项目列表（所有和用户个人）
- 项目删除
- 项目文件浏览
